2. Typography

Type scale from hero (2rem) to xs (0.6875rem). Font: Inter for UI, JetBrains Mono for numbers.

Hero — 2rem / 32px
--text-hero · Used for hero KPI values
XL — 1.5rem / 24px
--text-xl · Page titles
LG — 1.125rem / 18px
--text-lg · Section titles, KPI values
Base — 0.875rem / 14px
--text-base · Body text
SM — 0.8125rem / 13px
--text-sm · Compact body, table cells, buttons
XS — 0.6875rem / 11px
--text-xs · Labels, badges, captions
